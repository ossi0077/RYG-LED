# 3색 신호등 LED module

## 사양
	- 작동전압 : 5V
	- 작동전류 : 각 LED별 20mA

## 필요 하드웨어
	- 3색 신호등 LED module
	- Arduino UNO
	- UNO cable
	- F-M cable(4ea)

## 연결
	- RYG 단자의 경우 자신이 원하는 핀번호를 OUTPUT으로 설정하여 사용하시면 됩니다.(여기에선 D3, D4, D5를 사용)
	|3색 LED moudle|Arduino UNO|
	|--|--|
	|GND|GND|
	|R|D3(변경 가능)|
	|Y|D4(변경 가능)|
	|G|D5(변경 가능)|
